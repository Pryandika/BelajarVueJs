# Ini proyek git untuk coba buat website

## Instal dulu proyek ini ke PC. Buka command prompt, arahkan ke folder tempat proyek ini diinstal dan ketik:
```
npm install
```

### Kemudian ketik ini untuk menjalankan aplikasi:
```
npm run serve
```
